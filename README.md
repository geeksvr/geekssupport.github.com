# GeeksSupport - Gorilla Tag VR Community Website

Welcome to the GeeksSupport website for Gorilla Tag VR fan game enthusiasts! This website provides resources for unban appeals, community rules, and player reporting.

## Features

- **Unban Appeals**: Submit your appeal if you believe you were banned unfairly
- **Community Rules**: View our comprehensive rules for the community
- **Player Reporting**: Report players who violate community guidelines
- **Responsive Design**: Works on desktop and mobile devices

## Getting Started

To run this website locally:

1. Clone or download this repository
2. Open `index.html` in your web browser
3. The website will load with all functionality

## Deployment

To deploy this website to a live domain (geekssupport.com):

1. Purchase the domain name from a registrar (e.g., GoDaddy, Namecheap)
2. Choose a web hosting provider (e.g., Netlify, Vercel, GitHub Pages, traditional host)
3. Upload all files from this directory to your web server
4. Configure your domain's DNS settings to point to your hosting provider
5. Ensure your favicon.ico file is in the root directory for proper branding

## Files Included

- `index.html`: Main HTML file containing all website content
- `styles.css`: Stylesheet for the website's appearance
- `script.js`: JavaScript for interactive functionality
- `favicon.ico`: Website icon (you need to create this file)
- `robots.txt`: Instructions for web crawlers
- `sitemap.xml`: Site map for search engines
- `README.md`: This file

## Customization

Feel free to customize the website to better suit your community's needs:

- Modify colors in `styles.css`
- Update content in `index.html`
- Adjust animations in `script.js`
